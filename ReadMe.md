# 个人学习效率小助手

> 基于 Web 标准的前端综合实践项目 | 参赛主题：个人学习效率小助手 |
小组：硬核小玩家

## 项目概述
本项目旨在打造一个轻量化、无干扰的个人学习管理工具。项目严格遵循作业要求，在高质量完成所有基础功能（番茄钟、待办清单、数据统计、本地存储）的基础上，引入了数据可视化与用户激励体系，致力于通过技术手段提升学习专注度与成就感。

## 功能对标与实现
根据评审标准，本项目对基础功能的实现情况进行了严格自查与落实：

| 功能模块 | 作业要求 | 本项目实现情况 |
| :--- | :--- | :--- |
| **番茄钟** | 设置专注/休息时长，倒计时提醒 | **✅ 已实现**。支持三种模式切换（专注/短休/长休），具备自定义时长配置与自动连贯计时功能。 |
| **待办清单** | 增删改查，标记状态，日期排序 | **✅ 已实现**。支持任务分类筛选、实时状态更新，交互流畅。 |
| **时长统计** | 记录每日/每周时长，文字展示 | **✅ 超额完成**。不仅包含文字统计，还实现了**可视化图表**（柱状图/饼图），直观展示学习趋势。 |
| **数据持久化** | 关闭程序后数据不丢失 | **✅ 已实现**。基于 `LocalStorage` 构建本地数据库，确保待办、习惯、记录等数据永久保存。 |

## 创新与扩展亮点 (加分项)
为了提升用户体验并体现工程化思维，本项目在基础要求之外增加了以下扩展功能：

- **数据可视化分析**：摒弃了枯燥的纯文字展示，利用原生 DOM 操作动态渲染 SVG 图表，直观呈现“周学习趋势”与“科目时间分布”，体现了对数据的深度处理能力。
- **成就激励体系**：设计了一套完整的徽章系统，根据用户的学习时长和打卡习惯自动解锁成就，通过游戏化思维增强用户粘性。
- **多视图与导出功能**：支持生成学习周报/月报，并具备导出 PDF 与 SVG 图片的功能，方便用户进行线下复盘与分享。
- **沉浸式深色模式**：基于 CSS 变量实现了全局主题切换，适配不同光线环境，保护用户视力。

## 技术栈
- **核心逻辑**：Vanilla JavaScript (ES6+) - 采用模块化思维编写，逻辑与视图分离。
- **页面结构**：HTML5 - 语义化标签构建。
- **样式布局**：CSS3 - Flexbox 布局，CSS Variables 实现动态换肤。

本项目由团队成员协作开发，遵循模块化设计思想，明确划分了前端表现层与业务逻辑层。Git 提交记录完整反映了开发全过程，确保每位成员均实质性参与了核心编码工作：
